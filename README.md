# fifadict-capture
抓取泰服 FIFA Online3 数据库

[![js-standard-style](https://img.shields.io/badge/code%20style-standard-brightgreen.svg)](http://standardjs.com/)

## Install

```bash
  $ brew install phantomjs
  $ npm install casperjs -S
```

## Usage

```bash
  $ npm run id
  $ npm run player
```
